# Student Record Manager

A beginner-friendly Python GUI application to manage student records using Tkinter and MySQL.

## Features

- Add students (Name, Roll, Age, Gender, Address, Image)
- Edit and delete student records
- View records in a scrollable window
- Real-time updates after editing
- File chooser for image input

## Technologies Used

- Python
- Tkinter
- MySQL (via XAMPP)
- phpMyAdmin

## Setup

1. Start XAMPP and launch MySQL.
2. Create a database named `student_db` in phpMyAdmin.
3. Run the Python file to start the app.
